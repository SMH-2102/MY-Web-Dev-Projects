Introduction

C++ is a powerful general-purpose programming language.

It supports procedural, object-oriented, and generic programming.

Modern C++ includes powerful libraries and tools for development.

It is standardized by ISO and has evolved significantly over time.

C++ compilers are available for most platforms.

Hello World

The Hello World program is a classic beginner example.

#include <iostream>

Variables

Variables store data and must be declared with a type.

They can be initialized during declaration.

int age = 25; float pi = 3.14;

Data Types

C++ includes types like int, float, char, and bool.

  • int
  • float
  • char
  • bool
  • string

Example

int a=32; char name='X';
string name="Jacob"

Functions

Functions encapsulate reusable logic.

They have a return type, name, and parameters.

int add(int a, int b);
